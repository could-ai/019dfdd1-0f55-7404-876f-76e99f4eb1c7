# Maternal and Child Health (MCH) Follow-up System

A simple, structured MVP application to track pregnant women, new mothers, and children over time—making sure they receive the right care at the right stage.

## Features
- **Dashboard Overview**: See upcoming visits, missed visits, and total registrations.
- **Patient Registration**: Register mothers with details like Estimated Delivery Date (EDD).
- **Auto-Scheduling**: Automatically schedules ANC visits based on EDD.
- **Patient Detail & Visit Tracking**: Record new visits (ANC, Postnatal, Child) and mark scheduled visits as completed.
- **Local State MVP**: Runs purely locally in memory for easy demonstration.

## Tech Stack
- Framework: Flutter
- State Management: Provider (`ChangeNotifier`)

## How to Run
1. Ensure Flutter is installed.
2. Run `flutter pub get`
3. Run `flutter run`
